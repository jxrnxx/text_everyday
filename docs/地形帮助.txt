env_fog_controller（环境雾效控制器）
作用
控制整个地图的雾气效果，制造距离感和氛围。

关键参数
参数名称	作用	推荐值	说明
Fog Enable	启用雾效	✅ 勾选	总开关
Primary Fog Color	主雾颜色	(200, 220, 230)	淡青/白色有仙气感
Secondary Fog Color	次雾颜色	同主雾色	通常和主雾色一样
Fog Start	雾起始距离	1000-1500	距离这么远开始有雾
Fog End	雾结束距离	5000-8000	距离这么远完全雾化
Fog Max Density	最大雾密度	0.2-0.4	雾的浓度（0-1）

雾太浓，遮挡UI → 降低 Fog Max Density（0.2以下）
想要仙境感 → 用淡青色，密度 0.3-0.4
不想要雾 → 取消勾选 Fog Enable
 env_global_light（全局光照）
作用
整个地图的主光源（太阳光），决定整体亮度和阴影方向。 每个地图只需要1个！

关键参数
参数名称	作用	推荐值	说明
Color	光源颜色	(255, 240, 220)	暖白色自然
Light Scale	光照强度	1.5-3.0	就是亮度！
Ambient Angle	光照角度	(0, 100, 0)	控制阴影方向
Enable Shadows	启用阴影	✅ 勾选	让物体有阴影

 light_dynamic（动态点光源）
作用
局部照明，可以放多个，照亮特定区域。 适合：基地光源、底部光效、装饰灯光。

关键参数
参数名称	作用	推荐值	说明
Light color	光源颜色	按需设定	白色/金色/青色
Light brightness	光照强度	100-200	数值越大越亮
Maximum distance	光照范围	1500-2500	光能照多远
Inner angle	内圈角度	30	核心光锥角度
Outer angle	外圈角度	45	边缘光锥角度
Spotlight end radius	聚光半径	50-100	聚光灯模式用
